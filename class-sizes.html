A natural question is how crowded Cambridge’s classrooms are compared with nearby districts. The measure most people picture is class size — the number of students in a typical classroom. NYSED does publish an average class size for each district, and we look at it first; but it is reported only for a handful of specific grades and subjects (there is no single official number) and only since 2018–19. So for a longer, more complete view we also use K-12 students per teacher: a district’s K-12 enrollment divided by its number of teachers.

Students per teacher is a useful staffing comparison, but it is not class size, and it is always lower than class size — because it counts every teacher (including part-time teachers and specialists who don’t lead a regular class), not just the teachers standing in front of a full classroom. Read it as a staffing ratio, not a count of children in a room.

A reminder from the Overview: the teacher figure is a district-wide headcount (all grades, not full-time-equivalents), so this is an approximate measure for comparing districts and tracking change over time. Lower means fewer students per teacher.

1.1 Average class size

Start with the measure people actually picture. NYSED reports an average class size for each district — but only for a specific set of classes (the early elementary grades and the courses tied to State tests), and only since 2018–19. There is no single official “district class size,” so the line below is the median across all of a district’s reported classes each year: a robust middle value that a stray two-student elective can’t throw off.

Cambridge has the smallest typical class size of the group.

Code 
g = cc.class_size_median_for(cc.GRAPH_GROUP).drop_nulls("median_class_size")
cambridge = g.filter(pl.col("district") == cc.FOCUS)

(
    ggplot(g, aes("year_end", "median_class_size", color="district"))
    + geom_line(size=1)
    + geom_point(size=1.8)
    + geom_line(cambridge, size=2.4)  # emphasize the focus district
    + scale_color_manual(
        values=cc.PALETTE, name="", limits=cc.district_order(cc.GRAPH_GROUP)
    )
    + scale_x_continuous(breaks=list(range(2019, 2026)))
    + labs(
        x="School year ending",
        y="Median class size",
        # Title states only the facts; the interpretive headline lives above the
        # figure in the prose (see the project's graph-title convention).
        title="Class size",
    )
    + cc.theme_ccs()
)

Across these years Cambridge’s median class size runs about 11 to 15 students — consistently the lowest or near-lowest of the five, and several students below Schuylerville, Salem and Hoosick Falls. (Salem’s sharp dip in 2021 is a pandemic-year reporting artifact, not a real change.) That is the same story the staffing ratio tells next, seen at the level of actual classrooms: smaller classes, fewer students per teacher.

Two cautions before leaning on it too hard. NYSED measures class size for only a handful of classes, and the counts are small — a single combined section or a lightly-enrolled elective can swing a small district’s numbers from year to year. And the series only starts in 2018–19. For a longer, more complete view we turn to students per teacher — which exists for every district back to 2017–18, though, as the next section explains, it is a staffing ratio, not class size.

1.1.1 A sharper cut: grades 3–8 core classes

The median above mixes very different classes — small high-school electives with larger elementary sections — and is swayed by which classes each district happens to report. A cleaner, apples-to-apples comparison restricts to the twelve core academic classes every district teaches and reports: English and Math in grades 3 through 8. (NYSED relabeled these classes in 2021; the build unifies the old and new names.)

On the same twelve classes, Cambridge’s are the smallest in the group — by several students per class.

Cambridge’s grades 3–8 core classes run about 10 to 16 students, several below the other four districts in most years (Salem’s dip to 9 in 2021 is the same pandemic-reporting artifact). This is the most like-for-like class-size comparison the NYSED data supports; the broad median above and the students-per-teacher ratio below tell the same story with wider coverage.

1.1.2 High school: the Regents courses

High school is measured not by grade but as a set of specific Regents courses, and the picture changes. Sections are smaller and more variable — a capstone like Physics often runs 6–8 students — so we read it two ways: a clean gateway basket of the three courses nearly every student takes (Algebra I, English / ELA III, and Biology), and the median across all ten reported Regents courses.

At the high-school level the gap largely closes — Cambridge’s classes are middle-of-the-pack, not distinctly smaller.

1.3 Another view: teachers per 100 students

Students per teacher and teachers per 100 students are the very same data seen from opposite ends. The first asks how many students share each teacher; the second asks how many teachers a district employs for every 100 students. One is just the other turned upside-down — but flipping it puts staffing, rather than crowding, in the foreground, and now a higher line means more teachers per student.

Seen this way, Cambridge now employs the most teachers per 100 students of the group.

Code 
g = (
    cc.panel_for(cc.GRAPH_GROUP)  # 5 districts, 2018+
    .select("district", "year_end", "num_teachers", "k12_enrollment")
    .drop_nulls()
    .with_columns(
        (pl.col("num_teachers") / pl.col("k12_enrollment") * 100).alias(
            "teachers_per_100_students"

)
cambridge = g.filter(pl.col("district") == cc.FOCUS)

(

This is the mirror image of the first chart: wherever a district’s students-per-teacher ratio fell, its teachers-per-100-students rose. In 2017–18 only Salem employed more teachers per 100 students than Cambridge. Cambridge’s line then climbed — its K-12 enrollment fell about 15% while its teacher headcount held roughly steady — and by the two most recent years Cambridge has the most teachers per 100 students of the five (about 11.5 in 2024–25). The jump in reported teachers in 2023–24 pushed its 2024 figure especially high (12.2). The two larger districts, Schuylerville and Hoosick Falls, stay lowest, around 9 to 9.6 teachers per 100 students.

1.5 A national cross-check: NCES teacher FTE

The ratio above has two limitations worth testing. NYSED reports a teacher headcount, whereas the standard pupil-teacher ratio — the one NCES, the OECD and most cross-state comparisons use — divides enrollment by teacher full-time-equivalents (FTE). And it comes from a single state source. The federal Common Core of Data (CCD) offers an independent check: it reports teacher FTE and enrollment for every district. (CCD labels a school year by its fall term, so its years are shifted here to match NYSED’s school-year-ending labels.)

First the diagnostic question — do the two measures agree for Cambridge?

The two measures never differ by more than about two-thirds of a student per teacher, and both fall sharply in 2023–24.

Code 
ny = (
    cc.panel_for(cc.GRAPH_GROUP)
    .filter(pl.col("district") == cc.FOCUS)
    .select("year_end", pl.col("k12_students_per_teacher").alias("ratio"))
    .drop_nulls()
    .with_columns(pl.lit("NYSED (teacher headcount)").alias("source"))
)
ccd_cam = (
    cc.ccd_ratio_for(cc.GRAPH_GROUP)
    .filter((pl.col("district") == cc.FOCUS) & (pl.col("year_end") >= 2018))
    .select("year_end", pl.col("ccd_students_per_teacher").alias("ratio"))
    .drop_nulls()
    .with_columns(pl.lit("NCES (teacher FTE)").alias("source"))
)
both = pl.concat(
    [
        ny.with_columns(pl.col("year_end").cast(pl.Int64)),
        ccd_cam.with_columns(pl.col("year_end").cast(pl.Int64)),
    ]
)
source_colors = {
    "NYSED (teacher headcount)": "#C8102E",
    "NCES (teacher FTE)": "#2C6E9B",
}

(
    ggplot(both, aes("year_end", "ratio", color="source"))
    + geom_line(size=1.2)
    + geom_point(size=2)
    + scale_color_manual(values=source_colors, name="")
    + scale_x_continuous(breaks=list(range(2018, 2026)))
    + labs(
        x="School year ending",
        y="Students per teacher",
        title="Cambridge students per teacher: headcount vs. FTE",
    )
    + cc.theme_ccs()
)

The two lines stay close because, in small districts, a teacher headcount and its full-time-equivalent are nearly the same — there are few part-timers. More important, the independent FTE measure shows the same sharp decline in 2023–24, so that turning point is not an artifact of how NYSED counts teachers.

Having confirmed the two agree, the next chart and table repeat the earlier views using the NCES FTE-based ratio — the version directly comparable to other states and to national figures.

On the NCES measure too, Cambridge has the lowest ratio of the group in the two most recent years.

## Another view: teachers per 100 students

One is just the other turned upside-down — but flipping it puts staffing, rather
than crowding, in the foreground, and now a **higher** line means **more**
teachers per student.

**Seen this way, Cambridge now employs the most teachers per 100 students of the
group.**

## A national cross-check: NCES teacher FTE

The ratio above has two limitations worth testing. NYSED reports a teacher
*headcount*, whereas the standard pupil-teacher ratio — the one NCES, the OECD
and most cross-state comparisons use — divides enrollment by teacher
*full-time-equivalents (FTE)*. And it comes from a single state source. The
federal **Common Core of Data (CCD)** offers an independent check: it reports
teacher FTE and enrollment for every district. (CCD labels a school year by its
fall term, so its years are shifted here to match NYSED's school-year-ending
labels.)

First the diagnostic question — do the two measures agree for Cambridge?

**The two measures never differ by more than about two-thirds of a student per teacher, and both fall sharply in 2023–24.**

```{python}
#| label: fig-cambridge-sources
#| fig-cap: "Cambridge students per teacher: NYSED (teacher headcount) vs. NCES Common Core of Data (teacher FTE), by school year ending. Sources: NYSED Enrollment and Student & Educator databases; NCES CCD via the Urban Institute Education Data Portal."
ny = (
    cc.panel_for(cc.GRAPH_GROUP)
    .filter(pl.col("district") == cc.FOCUS)
    .select("year_end", pl.col("k12_students_per_teacher").alias("ratio"))
    .drop_nulls()
    .with_columns(pl.lit("NYSED (teacher headcount)").alias("source"))
)
ccd_cam = (
    cc.ccd_ratio_for(cc.GRAPH_GROUP)
    .filter((pl.col("district") == cc.FOCUS) & (pl.col("year_end") >= 2018))
    .select("year_end", pl.col("ccd_students_per_teacher").alias("ratio"))
    .drop_nulls()
    .with_columns(pl.lit("NCES (teacher FTE)").alias("source"))
)
both = pl.concat(
    [
        ny.with_columns(pl.col("year_end").cast(pl.Int64)),
        ccd_cam.with_columns(pl.col("year_end").cast(pl.Int64)),
    ]
)
source_colors = {
    "NYSED (teacher headcount)": "#C8102E",
    "NCES (teacher FTE)": "#2C6E9B",
}

(
    ggplot(both, aes("year_end", "ratio", color="source"))
    + geom_line(size=1.2)
    + geom_point(size=2)
    + scale_color_manual(values=source_colors, name="")
    + scale_x_continuous(breaks=list(range(2018, 2026)))
    + labs(
        x="School year ending",
        y="Students per teacher",
        title="Cambridge students per teacher: headcount vs. FTE",
    )
    + cc.theme_ccs()
)
```

The two lines stay close because, in small districts, a teacher headcount and its
full-time-equivalent are nearly the same — there are few part-timers. More
important, the independent FTE measure shows the **same sharp decline in
2023–24**, so that turning point is not an artifact of how NYSED counts teachers.
