A natural question is how crowded Cambridge’s classrooms are compared with nearby districts. The measure most people picture is class size — the number of students in a typical classroom — but NYSED does not publish class size by district in the data used here. The closest available measure is K-12 students per teacher: a district’s K-12 enrollment divided by its number of teachers.

Students per teacher is a useful staffing comparison, but it is not class size, and it is always lower than class size — because it counts every teacher (including part-time teachers and specialists who don’t lead a regular class), not just the teachers standing in front of a full classroom. Read it as a staffing ratio, not a count of children in a room.

A reminder from the Overview: the teacher figure is a district-wide headcount (all grades, not full-time-equivalents), so this is an approximate measure for comparing districts and tracking change over time. Lower means fewer students per teacher.

1.1 Students per teacher, over time

The chart below tracks the graph group from school-year-ending 2018 (the first year teacher counts exist) through 2025.

Cambridge now has the fewest students per teacher of the group.

Code 
g = (
    cc.panel_for(cc.GRAPH_GROUP)  # 5 districts, 2018+
    .select("district", "year_end", "k12_students_per_teacher")
    .drop_nulls()
)
cambridge = g.filter(pl.col("district") == cc.FOCUS)

(
    ggplot(g, aes("year_end", "k12_students_per_teacher", color="district"))
    + geom_line(size=1)
    + geom_point(size=1.8)
    + geom_line(cambridge, size=2.4)  # emphasize the focus district
    + scale_color_manual(
        values=cc.PALETTE, name="", limits=cc.district_order(cc.GRAPH_GROUP)
    )
    + scale_x_continuous(breaks=list(range(2018, 2026)))
    + labs(
        x="School year ending",
        y="K–12 students per teacher",
        # Title states only the facts; the interpretive headline lives above the
        # figure in the prose (see the project's graph-title convention).
        title="District K–12 enrollment ÷ teacher headcount, 2017–18 to 2024–25",
    )
    + cc.theme_ccs()
)

In 2017–18 only Salem had a lower ratio than Cambridge. Cambridge’s ratio then drifted down — its K-12 enrollment fell about 15% (from 869 to 741) while its teacher headcount stayed roughly flat — and by the two most recent years Cambridge has the lowest students-per-teacher ratio of the five. A jump in reported teachers in 2023–24 pushed its 2024 figure especially low (8.2). Schuylerville and Greenwich actually fell more over the full period (about 2.0 and 1.9, versus Cambridge’s 1.5), but from much higher starting points, so they remain above Cambridge.

1.2 The full comparison group

The table widens the lens to all twelve table-group districts. Columns are school-years-ending; the final column is the change from 2018 to 2025 (a negative value means fewer students per teacher than in 2018). Cambridge is highlighted.

Code 
t = cc.panel_for(cc.TABLE_GROUP)

# Long -> wide (years become columns), Washington County districts first,
# then alphabetical within each region.
wide = (
    t.select("region", "district", "year_end", "k12_students_per_teacher")
    .with_columns(
        pl.when(pl.col("region") == "Washington County")
        .then(0)
        .otherwise(1)
        .alias("_rank")
    )
    .pivot(
        values="k12_students_per_teacher",
        index=["_rank", "region", "district"],
        on="year_end",
    )
    .sort(["_rank", "district"])
    .drop("_rank")
)

year_cols = [c for c in wide.columns if c.isdigit()]
wide = wide.with_columns(
    (pl.col("2025") - pl.col("2018")).alias("chg")
)

(
    GT(wide, rowname_col="district", groupname_col="region")
    .tab_header(
        title="K–12 students per teacher",
        subtitle="School-year-ending 2018 (2017–18) through 2025 (2024–25)",
    )
    .fmt_number(columns=year_cols + ["chg"], decimals=1)
    .tab_spanner(label="School year ending", columns=year_cols)
    .cols_label(chg=md("Change<br>2018→2025"))
    .sub_missing(missing_text="—")
    .tab_style(
        style=[style.fill(color="#FCEFB4"), style.text(weight="bold")],
        locations=[
            loc.body(rows=pl.col("district") == cc.FOCUS),
            loc.stub(rows=pl.col("district") == cc.FOCUS),
        ],
    )
    .tab_source_note(
        md(
            "**Source:** NYSED Enrollment database (K-12 enrollment) and "
            "Student &amp; Educator database (teacher counts). "
            "Ratio = district K-12 enrollment ÷ teacher headcount; an "
            "approximate comparison measure, not official class size. "
            "Year = school-year-ending (2025 = 2024–25)."
        )
    )
)

Across the wider group, Cambridge ends among the lowest ratios — close to Salem — while the rest of Washington County’s K-12 districts span a wider range. The table is also a reminder that these districts differ greatly in size (compare small districts like Fort Edward with larger ones like Hudson Falls and Granville), which shapes how much a single hire or a small enrollment swing moves the ratio.

1.3 A national cross-check: NCES teacher FTE

The ratio above has two limitations worth testing. NYSED reports a teacher headcount, whereas the standard pupil-teacher ratio — the one NCES, the OECD and most cross-state comparisons use — divides enrollment by teacher full-time-equivalents (FTE). And it comes from a single state source. The federal Common Core of Data (CCD) offers an independent check: it reports teacher FTE and enrollment for every district. (CCD labels a school year by its fall term, so its years are shifted here to match NYSED’s school-year-ending labels.)

First the diagnostic question — do the two measures agree for Cambridge?

The two measures never differ by more than about two-thirds of a student per teacher, and both fall sharply in 2023–24.

Code 
ny = (
    cc.panel_for(cc.GRAPH_GROUP)
    .filter(pl.col("district") == cc.FOCUS)
    .select("year_end", pl.col("k12_students_per_teacher").alias("ratio"))
    .drop_nulls()
    .with_columns(pl.lit("NYSED (teacher headcount)").alias("source"))
)
ccd_cam = (
    cc.ccd_ratio_for(cc.GRAPH_GROUP)
    .filter((pl.col("district") == cc.FOCUS) & (pl.col("year_end") >= 2018))
    .select("year_end", pl.col("ccd_students_per_teacher").alias("ratio"))
    .drop_nulls()
    .with_columns(pl.lit("NCES (teacher FTE)").alias("source"))
)
both = pl.concat(
    [
        ny.with_columns(pl.col("year_end").cast(pl.Int64)),
        ccd_cam.with_columns(pl.col("year_end").cast(pl.Int64)),
    ]
)
source_colors = {
    "NYSED (teacher headcount)": "#C8102E",
    "NCES (teacher FTE)": "#2C6E9B",
}

(
    ggplot(both, aes("year_end", "ratio", color="source"))
    + geom_line(size=1.2)
    + geom_point(size=2)
    + scale_color_manual(values=source_colors, name="")
    + scale_x_continuous(breaks=list(range(2018, 2026)))
    + labs(
        x="School year ending",
        y="Students per teacher",
        title="Cambridge students per teacher: headcount vs. FTE",
    )
    + cc.theme_ccs()
)

The two lines stay close because, in small districts, a teacher headcount and its full-time-equivalent are nearly the same — there are few part-timers. More important, the independent FTE measure shows the same sharp decline in 2023–24, so that turning point is not an artifact of how NYSED counts teachers.

Having confirmed the two agree, the next chart and table repeat the earlier views using the NCES FTE-based ratio — the version directly comparable to other states and to national figures.

On the NCES measure too, Cambridge has the lowest ratio of the group in the two most recent years.
