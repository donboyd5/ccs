1  Demographics: Rural aging and population decline

  • Show All Code
  • Hide All Code
  • View Source

Cambridge is caught in a storm buffeting rural school districts throughout New York.

Population is declining in rural areas throughout New York.

The main reasons for this decline are:

  • Declines in the number of births, which is a statewide trend
  • Out-migration of younger adults, leading to an aging population
  • Economic factors that make rural areas less attractive for new residents

The charts below put Cambridge’s county — Washington County — alongside its neighbors on both sides of the state line. Each county’s population is shown relative to its own starting point (indexed to 100), so the lines compare trends, not sizes. Washington County is the red line in each chart.

**Cambridge is caught in a storm buffeting rural school districts throughout New York.**

The main reasons for this decline are:

-  Declines in the number of births, which is a statewide trend

```{python}
#| label: setup
#| include: false
import os
import sys
from pathlib import Path

sys.path.insert(0, os.getcwd())

import polars as pl
from plotnine import (
    aes,
    annotate,
    geom_line,
    geom_vline,
    ggplot,
    labs,
    scale_color_manual,
    scale_x_continuous,
    scale_y_continuous,
)
from great_tables import GT, loc, md, style

import comparisons as cc


def _find_repo_root() -> Path:
    cwd = Path.cwd()
    for base in [cwd, *cwd.parents][:5]:
        if (base / "data" / "crosswalks").is_dir():
            return base
    raise FileNotFoundError("Could not find repo root")


REPO_ROOT = _find_repo_root()
PROCESSED_DIR = REPO_ROOT / "data" / "processed"

pop = pl.read_parquet(PROCESSED_DIR / "demographics_county_pop.parquet")
components = pl.read_parquet(PROCESSED_DIR / "demographics_washington_components.parquet")

#: Short, state-tagged labels for the chart legends (VT counties tagged so a
#: reader sees the chart spans two states; NY counties are the default context).
COUNTY_LABEL: dict[str, str] = {
    "Washington": "Washington",
    "Warren": "Warren",
    "Saratoga": "Saratoga",
    "Rensselaer": "Rensselaer",
    "Bennington County": "Bennington (VT)",
    "Rutland County": "Rutland (VT)",
}
COUNTY_PALETTE: dict[str, str] = {
    "Washington": "#C8102E",      # bold red — Cambridge's county, the focus
    "Warren": "#2C6E9B",
    "Saratoga": "#3F9C5A",
    "Rensselaer": "#8156A7",
    "Bennington County": "#E08214",
    "Rutland County": "#6C757D",
}


def _indexed(df: pl.DataFrame, base_year: int) -> pl.DataFrame:
    """Add an ``index`` column = population / county's ``base_year`` value × 100."""
    base = (
        df.filter(pl.col("year") == base_year)
        .select("geoid", pl.col("population").alias("_base"))
    )
    return (
        df.join(base, on="geoid")
        .with_columns((pl.col("population") / pl.col("_base") * 100).alias("index"))
        .drop("_base")
        .with_columns(pl.col("geography").replace_strict(COUNTY_LABEL).alias("county"))
    )
```
